# FAQ

**What powers this site?**  
Jekyll 3.10, hosted on GitHub Pages, styled with a customized Indigo theme.

**How can I preview it locally?**  
Clone the repo, run `bundle install`, then `bundle exec jekyll serve` and browse http://localhost:4000.

**Can I reuse the theme?**  
Indigo is MIT licensed. Grab it from [the original repository](https://github.com/sergiokopplin/indigo) and tailor it to your needs. Make sure to give the original author a follow/star!
